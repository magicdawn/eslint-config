## Install

```sh
$ cnpm i -D @magicdawn/eslint-config
```

## Usage

### `default` for Node.js project

```yml
# .eslintrc.yml
extends:
  - @magicdawn
```

### `vue` for vue project

```yml
# .eslintrc.yml
extends:
  - @magicdawn/eslint-config/vue
```
